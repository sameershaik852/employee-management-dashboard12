# Employee Management Dashboard

## Project Overview
This project is a React-based Employee Management Dashboard built as part of an assessment.
It allows users to log in and manage employee records with full CRUD functionality.

---

## Tech Stack Used
- React (Vite)
- JavaScript (ES6)
- React Router DOM
- HTML5
- CSS3
- LocalStorage

---

## Steps to Run the Project Locally

### Prerequisites
- Node.js (v16 or higher)
- npm

npm install
